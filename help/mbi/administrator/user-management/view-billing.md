# 帐单信息

要查看、添加或更改帐户的计费信息，您必须拥有 `Billing` 权限。 [权限是逐个用户设置的](../../administrator/user-management/user-management.md) 创建用户帐户时。 如果您需要 `Billing` 权限，请联系您帐户的管理员。
