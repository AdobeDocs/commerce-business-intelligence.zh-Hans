# 账单信息

要查看、添加或更改您帐户的账单信息，您必须 `Billing` 权限。 [权限是按用户进行设置的](../../administrator/user-management/user-management.md) 创建用户帐户时。 如果您需要 `Billing` 权限，请联系您帐户的管理员。
