# 在更新周期后重建查询

您不必重新生成查询。 使用[`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md)创建的报告将像在传统`Report Builder`中创建的报告一样保存。 SQL图表的更新过程是相同的 — 更新数据后，图表中的值将重新计算并重新显示。
