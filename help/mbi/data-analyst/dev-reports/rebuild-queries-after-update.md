# 更新周期后重建查询

您不必重新构建查询。 使用创建的报表 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 与传统 `Report Builder`. SQL图表的更新过程完全相同 — 更新数据后，将重新计算并重新显示图表中的值。
