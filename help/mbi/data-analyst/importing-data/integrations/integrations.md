# 使用集成连接外部数据

集成是数据源。 集成是可以连接到data warehouse的应用程序和数据库。

在此部分中，我们将介绍可用于将各种外部数据源连接到 [!DNL MBI].

要删除连接或集成，请联系 [支持](../../../guide-overview.md) 以及包含要删除的连接或集成的名称。
