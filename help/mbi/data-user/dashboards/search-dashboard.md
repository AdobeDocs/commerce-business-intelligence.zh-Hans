# 搜索功能板

在本文中，我们将向您展示如何使用 [`Global Search` 功能](#global) 查找功能板以及如何搜索 [其他用户拥有的功能板](#other).

## 全局搜索 {#global}

的 `Global Search` 菜单，可搜索并选择要查看的功能板。

* *查看现有功能板的列表*，单击功能板。

* *搜索功能板*，单击功能板的下拉菜单后，在搜索栏中键入一些搜索标准。 如果有任何功能板与标准匹配，则它们将首先在列表中显示。

示例：

![功能板全局搜索](../../assets/dboard-global-search.gif)

## 查找其他用户拥有的功能板 {#other}

是否正在查找其他用户拥有的功能板？ 如果功能板可供其他人查看，您可以通过单击 **[!UICONTROL Find]** 在 `Dashboard Options` 下拉列表。

![查找功能板](../../assets/find-dboards-other-owners.png)
