# 从功能板中删除图表

要从功能板中删除图表，请单击齿轮(![](../../assets/gear-icon.png))图标，然后单击 **[!UICONTROL Remove from Dashboard]**.

>[!NOTE]
>
>删除图表的方式与不同 [正在删除它](../../data-user/dashboards/delete-chart.md). 另外， [可以将图表读入功能板](../../data-user/dashboards/add-charts-dashboard.md) 随时。
