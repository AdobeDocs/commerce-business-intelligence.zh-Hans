# 访问共享报告面板

让新用户快速上手很简单。

## 新用户

新用户创建密码后，将显示常用共享功能板列表。 他们可以选择要显示在仪表板上的仪表板，然后单击&#x200B;**[!UICONTROL Done]**。

## 现有用户

若要再次查看共享功能板的列表，用户可以单击&#x200B;**[!UICONTROL Find]**&#x200B;菜单中的`Global Search`。
