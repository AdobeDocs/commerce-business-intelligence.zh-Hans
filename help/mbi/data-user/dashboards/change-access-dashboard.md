# 更改对共享仪表板的访问权限

可以随时更改对功能板的访问权限，其步骤与 [最初共享功能板](../../data-user/dashboards/share-dashboard-with-users.md){： target=&quot;_blank&quot;}. 显示在中的用户列表中 `Share Dashboard` 菜单，根据需要更新用户/组织权限，然后单击 **[!UICONTROL Save Changes]** 以进行更新。
