# 更改对共享功能板的访问权限

可使用相同的步骤随时更改对仪表板的访问权限，以便最初[共享仪表板](../../data-user/dashboards/share-dashboard-with-users.md){: target="_blank"}。 在`Share Dashboard`菜单中显示的用户列表中，根据需要更新用户/组织权限，然后单击&#x200B;**[!UICONTROL Save Changes]**&#x200B;进行更新。
