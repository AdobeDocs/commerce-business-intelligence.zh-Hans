# 保留（取消共享）功能板

更换团队？ 在春天清洗你的[!DNL Commerce Intelligence]帐户吗？ 若要离开仪表板或取消共享您自己的仪表板，请在离开仪表板时单击屏幕顶部的&#x200B;**[!UICONTROL Shared]**。 单击&#x200B;**[!UICONTROL Leave this Dashboard]**&#x200B;以自行删除。
