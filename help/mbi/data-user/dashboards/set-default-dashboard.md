# 设置默认功能板

默认功能板是您打开时看到的第一个功能板 [!DNL MBI].

1. 在功能板中，单击 **[!UICONTROL Dashboard Options]** 位于屏幕顶部。

1. 单击 **[!UICONTROL Make Default]** 中。

1. 显示确认提示时，单击 **[!UICONTROL Yes]** 更改默认功能板。

这是您新的默认功能板。
