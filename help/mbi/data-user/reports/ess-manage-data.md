# 管理数据

通过“管理数据”，可以访问各种用于管理集成、报告和图表数据、功能板和导出的工具。

## 要访问 [!DNL Manage Data]：

1. 在菜单中，单击 **[!DNL Manage Data]**.

1. 在侧栏中，选择以下标题下需要的主题：

   * `Data Sources`
   * `Manage Data`
   * `Export Data`
