# 创建报告副本

通常，您会发现自己希望创建定义相似的图表，而您只想更改其中一个元素 — 例如编辑过滤器或更改 `group by`.

在这些情况下，请启动 `Chart Editor` 并单击 **[!UICONTROL Save As]** 在右上角。 这将复制您现有的图表，并使用您选择的新名称将其添加到当前功能板，并允许您立即编辑新图表的设置。
