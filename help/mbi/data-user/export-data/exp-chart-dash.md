# 导出图表数据

如果要查看图表后面的数据，请单击任意图表右上角的齿轮图标。

选择：

- `Report Export`
- `Full .csv Export`
- `Full Excel Export`
- `Raw Export`

您还可以定义 [表和列](../../tutorials/export-raw-data.md).

如果需要在演示文稿或文档中包含图表，可以通过单击任意图表右上角的齿轮图标，将图表导出为PNG、PDF或SVG格式的图像。 选择 `Enlarge` 选项，然后单击 **[!UICONTROL Download]** 以完成导出。
