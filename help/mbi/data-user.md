# 数据用户

您可以使用数据来帮助制定业务决策。 您可能从团队的数据分析师那里收到报告和仪表板，但您还需要了解如何创建这些报告和仪表板。

本节将演示如何：
* 在[!DNL Commerce Intelligence]中使用报告和仪表板获取和显示数据
* 通过电子邮件和`.csv`文件与利益相关者共享该数据
